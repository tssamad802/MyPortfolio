

# Portfolio Website for Abdul Samad Ahmed Khan

## Overview
A sleek, modern personal portfolio website with dark/light mode, smooth animations, and a professional yet creative aesthetic — all built with a single font family (Inter) and consistent design language.

---

## Sections

### 1. Header / Navbar
- Logo displaying "Abdul Samad Ahmed Khan"
- Navigation links: Home, About, Skills, Experience, Projects, Contact
- Dark/Light mode toggle with smooth transition
- Mobile hamburger menu with slide-in animation
- Sticky navbar that subtly changes on scroll

### 2. Hero Section
- Large animated heading with your name
- Tagline: "Aspiring Software Engineer | Junior Web Developer | AI Enthusiast"
- Animated "View My Work" button that scrolls to Projects
- Subtle particle/gradient background animation
- Fade-in entrance animations

### 3. About Section
- Brief bio about being a CS first-year student passionate about web development and AI
- Clean layout with subtle background gradient animation
- Fade-in-on-scroll effect

### 4. Skills Section
- Skill cards with icons for: HTML, CSS, JavaScript, PHP, MariaDB, WordPress, Elementor, AJAX, AI Concepts
- Hover effects that lift and highlight cards
- Animated progress bars or visual indicators
- Staggered entrance animations on scroll

### 5. Experience Section
- Timeline-style layout with three entries:
  - **Rootlet Solutions**, Karachi — 4 months internship
  - **Israin Solutions**, Hyderabad — 3 months internship
  - **Gexton**, Hyderabad — Current internship
- Cards with hover highlight effects
- Scroll-triggered animations

### 6. Projects Section
- Interactive project cards with hover overlay revealing details
- Placeholder projects (easily customizable with real project data later)
- Consistent card design with hover scale and shadow effects

### 7. Contact & Social Links
- Email, LinkedIn, GitHub, YouTube (CodeCrase), portfolio link
- Animated social link buttons with hover effects
- Simple contact form layout

---

## Design & UX
- **Single font family** (Inter) across the entire site
- **Dark/Light mode** toggle with smooth color transitions
- **Smooth scrolling** between all sections
- **Scroll-triggered animations** for all sections (fade-in, slide-up)
- **Hover effects** on all interactive elements (buttons, cards, links)
- **Subtle background animations** (gradient shifts)
- **Fully responsive** — desktop, tablet, and mobile
- **Consistent button styling** throughout
