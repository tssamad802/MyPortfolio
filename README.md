## 📖 What Is This Project?

This is my **personal portfolio website** — a professional web page that acts like my online resume and project showcase. Anyone who visits this site can:

- 👀 Learn **who I am** and what I do
- 💼 View my **projects** with descriptions and links
- 🛠️ See the **skills and technologies** I work with
- 📬 **Contact me** easily

## 🚀 Getting Started — Run Locally in 4 Steps

> **Prerequisites:** Make sure you have **Node.js** and **npm** installed.
> 👉 Don't have them? [Install Node.js here](https://nodejs.org/)

### Step 1 — Clone the Repository

```bash
git clone https://github.com/tssamad802/MyPortfolio.git
```

### Step 2 — Go into the Project Folder

```bash
cd MyPortfolio
```

### Step 3 — Install Dependencies

```bash
npm install
```

### Step 4 — Start the Development Server

```bash
npm run dev
```

🎉 Done! Open your browser and go to → **`http://localhost:5173`**

## 🌐 Deployment

This project can be deployed easily with platforms like:

- **[Vercel](https://vercel.com/)** → Recommended, free & fast
- **[Netlify](https://netlify.com/)** → Drag and drop or GitHub connect
- **[GitHub Pages](https://pages.github.com/)** → Free hosting from GitHub

### Deploy to Vercel (Easiest Way)

1. Push your code to GitHub ✅
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"New Project"** and import this repo
4. Hit **Deploy** — that's it! 🚀

## ✏️ How to Customize This Portfolio

Want to make it your own? Here's what to change:

- **Your Name & Bio** → Edit the Hero/About section component in `src/components/`
- **Your Projects** → Update the projects data/component with your own projects
- **Your Skills** → Edit the skills section with your technologies
- **Colors & Theme** → Update `tailwind.config.ts` to change the color palette
- **Profile Photo** → Replace the image in the `public/` folder
